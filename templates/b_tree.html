Pseudocode

1. Insert key into leaf node
2. Sort keys in node
3. If node full (≥ M keys) → split
4. Promote middle key to parent

B-Tree Properties

📦 Each node has at most M−1 keys
🌿 All leaves at same depth level
🔀 Non-leaf nodes have ceil(M/2) min children
⬆ Split propagates median key upward
Commonly used in databases & file systems
